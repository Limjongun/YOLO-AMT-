Adaptive Memory Trajectory YOLO (AMT-YOLO)

Overview

AMT-YOLO adalah konsep arsitektur berbasis YOLO yang menambahkan tiga
modul utama: 1. Adaptive Resolution 2. Temporal Memory Module 3.
Trajectory Prediction Head

Tujuan: - Mempertahankan inference real-time - Meningkatkan stabilitas
deteksi pada video - Memperbaiki tracking saat occlusion - Memprediksi
lintasan objek beberapa frame ke depan

Module Description

1. Adaptive Resolution Router

Input: - Current frame - Scene complexity estimation

Output: - 640×640 (simple scene) - 768×768 (medium scene) - 1024×1024
(complex scene)

Purpose: - Mengurangi komputasi pada scene sederhana. - Meningkatkan
detail untuk objek kecil.

------------------------------------------------------------------------

2. YOLO Backbone

Fungsi: - Feature extraction - Multi-scale feature learning

Contoh backbone: - YOLO11 Backbone - CSP/C2f Block - SPPF

Output: - Multi-scale feature maps

------------------------------------------------------------------------

3. Temporal Memory Module

Input: - Current feature map - Previous memory feature

Fungsi: - Menyimpan informasi frame sebelumnya. - Mengatasi occlusion. -
Mengurangi flickering. - Memperkuat tracking.

Implementasi yang memungkinkan: - ConvGRU - ConvLSTM - Memory Bank -
Lightweight Transformer Memory

Output: - Memory-enhanced feature

------------------------------------------------------------------------

4. Adaptive Feature Fusion Neck

Fungsi: - Menggabungkan feature dari berbagai skala. - Memberikan bobot
adaptif terhadap feature penting.

Implementasi: - PAN - FPN - Attention Fusion

Output: - Rich feature representation

------------------------------------------------------------------------

5. Detection Head

Output: - Bounding Box - Class - Object Confidence - Tracking Embedding

Loss: - IoU Loss - DFL - Classification Loss

------------------------------------------------------------------------

6. Trajectory Prediction Head

Input: - Current object embedding - Memory embedding

Output: - Future center point - Future bounding box - Motion vector

Prediction Horizon: - 3–10 frame ke depan.

Loss: - Smooth L1 - MSE - Trajectory Loss

------------------------------------------------------------------------

Complete Pipeline

Frame ↓ Adaptive Resolution ↓ YOLO Backbone ↓ Temporal Memory ↓ Feature
Fusion Neck ↓ Detection Head ├── Detection ├── Tracking └── Trajectory
Prediction ↓ Output

Programming - Python 3.12+

Deep Learning - PyTorch - Ultralytics YOLO

Computer Vision - OpenCV - NumPy

Training - CUDA - cuDNN - Automatic Mixed Precision (AMP)

Tracking - ByteTrack - BoT-SORT

Memory Module - ConvGRU - ConvLSTM - TorchScript Module

Trajectory - Lightweight MLP - GRU - Temporal Transformer

Dataset - COCO - MOT17 - MOT20 - BDD100K - VisDrone

Evaluation Detection: - mAP@50 - mAP@50-95

Tracking: - HOTA - MOTA - IDF1

Trajectory: - ADE - FDE

Performance: - FPS - Latency - GFLOPs - Parameter Count

Research Contributions

1.  Adaptive Resolution untuk efisiensi komputasi.
2.  Temporal Memory untuk stabilitas deteksi video.
3.  Trajectory Prediction untuk prediksi lintasan objek.
4.  Integrasi ketiga modul dalam satu arsitektur end-to-end berbasis
    YOLO dengan fokus menjaga performa real-time.
